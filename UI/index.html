How to use Eye of Things


Things you'll need:

  • An AsBase Account
    		•  AsBase
  • An IFTTT Account
    		•
  • An IFTTT Recipe which pushs into the Eye of Things Databin
    		•

    Step 1: Once logged into IFTTT, create a new recipe. In this example we're going to use the 'Space' channel to create a trigger.

    Step 2: We select the ISS trigger.










    Step 3: We select a location to track.






















    Step 4: Select the Gmail Channel.













    Step 5: Send data to the Eye Of Things databin using these credentials, along with data that follows the Eye of Things data conventions. A description of our data schema can be found at EyeofThingsBerkeley.Github.Io
